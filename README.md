# 🚂 Railway Ticket System (ADBMS Project)

This is a database management system laboratory project built using Python (Flask) and PostgreSQL. The user interface layout and color theme are designed to match the official Bangladesh Railway website, with some modern CSS styling.

## 🛠️ Technology Stack

* **Frontend:** HTML5, CSS3, JavaScript
* **Backend:** Python Flask Framework
* **Database:** PostgreSQL (3NF Normalization)
* **Database Hosting:** Neon.tech Cloud

---

## 🚀 Main Features

1. **Bangladesh Railway Layout:** Matches the official portal's main components and structure.
2. **Interactive Seat Selection:** Users can view the train carriage and select specific seats manually.
3. **Structured 3NF Schema:** Database tables are properly normalized to avoid data redundancy (`database_schema_3nf.sql` file uploaded).
4. **Backend Operations:** Handles live calculations for ticket pricing, seat availability, and customer details.

## 🚀 Key Features
1. **Bangladesh Railway Theme:** Strict alignment with the official portal's color palette and component structuring.
2. **Interactive Seat Map:** Live train carriage visualization for seamless seat selection.
3. **Third Normal Form (3NF):** Fully structured SQL schema optimized to reduce data redundancy.
