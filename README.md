# 🚂 Smart Railway Ticketing Management System

A modern, responsive frontend project for a Railway Ticketing System developed as part of the **Advanced Database Management System (ADBMS)** course lab.

##  Features
* **Interactive Seat Map:** Select seats from AC and Non-AC coaches.
* **Business Logic:** Maximum booking limit of 4 seats per passenger.
* **Dynamic Pricing:** Automatic fare calculation based on coach type.
* **Digital Ticket:** Instant E-ticket generation for passengers after confirmation.
* **Admin Dashboard:** Password-protected panel (Pass: 123) to monitor sales, revenue, and passenger data.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure.
* **CSS3:** Modern UI with flexbox and grid.
* **JavaScript (ES6):** Dynamic seat generation, validation, and state management.

## 📂 Project Structure
* `index.html` - The main interface.
* `style.css` - Custom styling and animations.
* `script.js` - Core logic and functionality.
* `/v2` - Updated version with Route and Time management.
